# TRIAL RAxML using core_gene_alignment_filtered.aln (panaroo's output for more accurate tree)

1. ML tree inference
$ raxmlHPC-PTHREADS-AVX2 -safe -m GTRGAMMA -f d -p 12345 \
    -s /mnt/d/cap_adult_rscm/outputs/result_panaroo_clean_mode_strict/core_gene_alignment_filtered.aln \
    -n 1_output_tree \
    -w /mnt/d/cap_adult_rscm/outputs/result_raxml_from_panaroo \
    -T 14 # threads

2. Bootstrap 1000 times requires 2.150371 h * 1000/24 = 90 days. I turned it down to 100 iterations instead.
$ raxmlHPC-PTHREADS-AVX2 -safe -m GTRGAMMA -f d -p 12345 \
    -s /mnt/d/cap_adult_rscm/outputs/result_panaroo_clean_mode_strict/core_gene_alignment_filtered.aln \
    -n 2_bootstrap \
    -w /mnt/d/cap_adult_rscm/outputs/result_raxml_from_panaroo \
    -b 67890 -# 100 \
    -T 14 # threads

3. Combine bootstrap trees with the best ML tree
$ raxmlHPC-PTHREADS-AVX2 -safe -m GTRGAMMA -f b -p 12345 \
    -t /mnt/d/cap_adult_rscm/outputs/result_raxml_from_panaroo/RAxML_bestTree.1_output_tree \
    -z /mnt/d/cap_adult_rscm/outputs/result_raxml_from_panaroo/RAxML_bootstrap.2_bootstrap \
    -n 3_final_tree.nwk \
    -w /mnt/d/cap_adult_rscm/outputs/result_raxml_from_panaroo \
    -T 14 # threads
